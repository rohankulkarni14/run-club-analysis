# Run Club Analytics Dashboard

## 📌 Overview
This project analyzes participation and performance data from a running club with 100+ members to identify trends and improve event planning.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Excel

## 📊 Key Analysis
- Weekly participation trends
- Average distance and pace analysis
- New member growth over time

## 📈 Insights
- Participation showed a steady increase over time
- Peak attendance observed in later weeks
- Average pace improved as consistency increased

## 📎 Output
- Line chart showing participation trends
- Bar chart for new members

## 🚀 Conclusion
The analysis helps optimize event planning and improve community engagement.
